Este proyecto es la resolución del siguiente examen:
Realización de un pequeño proyecto con un formulario de login, que permitiera el
login del usuario y una vez logueado se mostrará un pequeño crud donde se 
listaran los usuarios que hay en el sistema y cualquier usuario logueado pudiera añadir usuarios, 
y modificar o eliminar los usuarios ya existentes. 
Para esta prueba usamos simplemente PHP, JS y CSS, sin otros frameworks o
librerías del lenguaje.

Instalación:
-Para la implementación de dicho proyecto es necesario tener un servidor apache y 
un servidor mysql.
-Crear una base de datos en mysql con el nombre "user", luego importar desde la nueva base de datos
el script "user.sql" que se encuentra dentro de la carpeta "examen", otra opción es ejecutar el contenido 
del archivo (user.sql) como consulta SQL dentro de la base de datos. 
-Mover la carpeta "examen" al directorio de aplicaciones de apache.
-Por último desde un browser se llama a la aplicación de la siguiente forma 
 http://[nombre del servidor]/[directorio de la aplicación].
 -el usuario por defecto para ingresar al sitio por primera vez (si no se crea una cuenta) es USUARIO: admin@admin.com CONTRASEÑA: admin.
